# 삼진엘앤디 디지털 전환(DX) 프로젝트 진행 계획서

- **작성일**: 2026. 6. 10.
- **작성**: 크레플(CREFLE Inc.)
- **버전**: v1

---

## 1. 프로젝트 개요

삼진엘앤디 제조팀에서 종이 문서와 개별 엑셀 파일로 관리하고 있는 현장 정보를 디지털화·표준화하고, 수집된 데이터를 열람(조회)하고 표준화된 보고서로 생성할 수 있는 시스템을 구축합니다.

- **마감 목표일**: 2026년 7월 31일(금)
- **최종 산출물**: 삼진엘앤디 동탄 본사에 시스템 설치 및 운영 교육 완료

## 2. 개발 범위 (기능)

| 구분 | 내용 |
| --- | --- |
| 정보 입력의 디지털화 | 종이 문서로 기록되는 정보를 컴퓨터 또는 휴대용 디바이스를 이용해 입력할 수 있도록 개발 |
| 디지털 정보 표준화 | 개별 엑셀 파일과 같이 데이터 분석에 정형화되어 있지 않은 정보를 표준화 모델링하고, 컴퓨터 또는 휴대용 디바이스로 입력할 수 있도록 개발 |
| 정보 열람 및 보고서 생성 | 수집된 데이터를 열람(조회)하고, 표준화된 형식의 보고서로 생성 |

## 4. 삼진엘앤디 협조 요청 사항

| 항목 | 내용 |
| --- | --- |
| 데이터 수집 | 종이 문서 및 엑셀 파일로 관리되는 디지털화 대상 자료 수집 |
| 업무 프로세스 정리 | 현재 데이터의 처리 과정을 최대한 상세히 정리 |
| 보고서 양식 정의 | 데이터 관련 보고서 형식 정의. 최소한 "어떤 내용이 포함된 어떤 보고서가 필요한지"는 정리 필요 (디자인 형식은 크레플에서 제시 가능) |
